# CLAUDE.md — Portfolio Дмитрия Прокофьева

## Описание проекта
Персональный сайт-портфолио. Одностраничный лендинг на Next.js с тёплой кремово-золотой эстетикой.

## Стек
- Next.js 16 + TypeScript
- Tailwind CSS
- Framer Motion (анимации)
- Google Fonts: Cormorant Garamond (заголовки) + DM Sans (текст)
- Vercel (хостинг)

## Структура файлов
```
components/
├── Hero.tsx          — главный экран
├── LogosStrip.tsx    — бегущая строка инструментов
├── Services.tsx      — услуги с офферами по Хормози
├── HowItWorks.tsx    — 3 шага процесса
├── Cases.tsx         — кейсы с цифрами
├── Contact.tsx       — контакты (тёмный фон)
└── Footer.tsx        — подвал
```

## Цветовая система (CSS переменные)
```
--cream          #F5F0E8   основной фон
--cream-dark     #EDE7D9   фон секций
--cream-deeper   #E0D8C8   разделители
--ink            #1A1612   основной текст
--ink-light      #3D3530   вторичный текст
--gold           #B8933F   акцент
--gold-light     #D4A853   акцент hover
--muted          #7A6E65   приглушённый текст
```

## Правила разработки
- Не менять цветовую схему без явного запроса
- Шрифты только через CSS переменные и Google Fonts в layout.tsx
- Все анимации через Framer Motion
- Тексты офферов написаны по системе Хормози — менять осторожно
