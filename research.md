## Экспертная оценка

### 👁 UX-дизайнер

**Что работает хорошо:**
- Чистая иерархия: имя → оффер → подтверждение цифрами → услуги → процесс → кейс → контакт
- Хорошая типографика — Cormorant Garamond даёт премиальный вид
- Анимации ненавязчивые, не отвлекают
- Matrix Rain — запоминающийся элемент, соответствует нише

**Ошибки и что бесит пользователей:**

1. **Нет навигации** — пользователь не понимает сколько контента на странице и где он находится. Фиксированный header с якорями (#услуги, #кейсы, #контакт) обязателен
2. **Нет кнопки "наверх"** — на длинной странице пользователь теряется
3. **Имя уходит за край** на некоторых разрешениях (уже частично фиксировали)
4. **Нет фото** — люди покупают у людей. Аватар или фото повышает доверие в разы
5. **Один кейс** — недостаточно для убеждения. Нужно минимум 3
6. **CTA только в Hero и Contact** — в секции услуг нет призыва к действию после каждой карточки
7. **Кнопки не выглядят как кнопки** на некоторых устройствах — нет очевидного hover-состояния для тач-экранов
8. **Нет якорей на секции** — ссылка "Смотреть кейсы" ведёт на #cases, но остальные секции без id
9. **Нет индикатора загрузки** при первом открытии — шрифты грузятся из Google, может быть FOUT (мигание шрифта)

**Обязательно должно быть:**
- [ ] Фиксированная навигация (header)
- [ ] Фото / аватар
- [ ] Кнопка "наверх"
- [ ] id на каждой секции
- [ ] Favicon (сейчас стандартный Next.js)

### 📈 Маркетолог

**Что работает хорошо:**
- Офферы написаны по системе Хормози — конкретика, цифры, гарантии
- Кейс с реальными числами (85%, 50ч) — это конвертирует
- "Первый созвон бесплатно" — снижает барьер входа
- Гарантии на каждую услугу — снимают риск

**Что повысит конверсию:**

1. **Нет социального доказательства** — отзывы, логотипы клиентов, скрины переписок. Это #1 триггер доверия в B2B
2. **Нет конкретной цены или вилки** — "от X до Y ₽" или "от X ₽". Клиент без ценника сразу думает "дорого" и уходит
3. **Слабый CTA в Contact** — "Написать в Telegram" слишком общий. Лучше: "Обсудить автоматизацию → бесплатный созвон 30 минут"
4. **Нет срочности и дефицита** — по системе Хормози это усилители оффера. Например: "Беру 3 новых проекта в апреле"
5. **Нет секции "Для кого"** — сейчас не понятно кому это подходит. Малый бизнес? Онлайн-школы? Рестораны?
6. **Статистика в Hero без контекста** — "85% сообщений обрабатывает ИИ" — это из одного кейса. Добавить подпись "кейс: косметолог, Петрозаводск"
7. **Нет FAQ** — снимает возражения до созвона. Типичные вопросы: "Сколько стоит?", "Как долго?", "Что если не понравится?", "Нужен ли технический специалист с моей стороны?"
8. **Email в контактах** — люди в 2025 не пишут на email незнакомым. Оставить только Telegram или добавить форму
9. **Нет UTM-аналитики** — не понятно откуда приходят люди

**Приоритет по конверсии:**
1. Добавить отзывы / соцдоказательства
2. Добавить FAQ
3. Усилить CTA
4. Добавить "Для кого"

### 💬 Telegram-разработчик (оценка интеграции)

**Что работает хорошо:**
- Прямая ссылка `t.me/DmitryiP03` — простейший и самый надёжный способ
- Не требует никакой инфраструктуры, работает всегда

**Ограничения текущего подхода:**

1. **Нет автоответа** — человек написал, ждёт. Если ты недоступен — он уходит. Нужен бот с автоприветствием
2. **Нет квалификации** — все пишут подряд, теряешь время на нецелевых. Бот должен задать 3 вопроса (ниша, задача, бюджет) до того как ты увидишь сообщение
3. **Нет уведомления о заявке** — если сделать форму на сайте, n8n может отправлять уведомление тебе в TG с данными клиента
4. **Нет CRM** — все обращения теряются в личке. Нужно хотя бы простое логирование в таблицу

**Что лучше не делать:**
- Не ставить ссылку на канал вместо личного аккаунта — люди хотят написать человеку, а не подписаться
- Не использовать длинный username с цифрами — сложно запомнить (@DmitryiP03 норм, но лучше было бы без цифр)
- Не добавлять QR-код на сайт — на десктопе бесполезно

**Что сделать в первую очередь:**
- [ ] Создать простого бота-автоответчика с квалификацией
- [ ] Подключить n8n: форма на сайте → уведомление тебе в TG
